# Wheel Support Assembly

This project demonstrates the complete design workflow of a wheel support assembly using SolidWorks. The project includes part modelling, assembly creation, engineering drafting, sectional views, and bill of materials (BOM) generation.

## Skills Demonstrated
- 3D Part Modelling
- Assembly Design
- Engineering Drawing Creation
- Section Views
- Bill of Materials (BOM)
- Technical Documentation

## Software Used
- SolidWorks 2025

## Components
- Base Plate
- Support Plates
- Wheel
- Bushing
- Shoulder Screw
- Bolts, Nuts and Washers

## Output
- Fully assembled CAD model
- Manufacturing drawing sheet
- BOM documentation
- Orthographic and isometric views
